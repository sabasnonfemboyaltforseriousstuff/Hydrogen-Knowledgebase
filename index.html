Frequently Asked Questions
How to download Hydrogen?

Installation

  1. Run the installation script
    • Copy the script from this channel
    • Open the Terminal app
    • Paste the script in and press Enter
  2. Enter your password
    • Type in your Mac password
    • Keep in mind that your password won't be visible on screen as you type (this is a security feature)
    • Press Enter after typing your password
  3. Wait for completion
    • The script will automatically download and install Hydrogen
    • It will also be reinstall Roblox
    • After the installation a new app called “Hydrogen-M” will be installed

Usage

  1. First steps
    • Open Roblox.
    • A popup will appear asking for your Hydrogen key
    • Click “Get key” or paste in your key if you already have one
  2. Paste in your script
    • Open the Hydrogen app
    • Paste the script you want to execute into the app
  3. Execute the script
    • Join any Roblox game
    • Switch back to the Hydrogen-M app
    • Click "Run Script"

If you get any errors during installation contact a member of the support team or make a ticket.

How to get a key?

To get a key you will need to go here and complete a few offers to get a key. There are many different offers so I have narrowed them down into three different categories instead of making a tutorial for EVERY offer.

App Downloads

  1. Click the download button and stay on the download page for 45 seconds
  2. Go back to the Work.ink tab after 45 seconds
  3. Remove the downloaded file

Browser Extensions

  1. Install the extension
  2. Go back to the Work.ink tab
  3. Remove the extension after step completion

Sign ups

  1. Click the sign up button
  2. Go to their website and stay there for 45 seconds
  3. Go back to the Work.ink tab after 45 seconds

Extra

  • Keep in mind that if you go back to the Work.ink tab before the 45 second timer is over you WILL have to start over again
  • Disable all ad blockers or any extensions that may interfere with Work.ink
  • This shouldn’t take longer than 5 minutes (if it does you’re probably doing it wrong :3)
My script doesn't work or crashes.

Not all scripts are compatible with Hydrogen. If a script doesn't work, it's either outdated or not compatible with Hydrogen. You can report non-functional scripts here.

How to auto-execute?

To auto-execute a script, open TextEdit, create a new document, and paste your script into it. Save it by pressing CMD + S and move it to the "autoexecute" folder within your Hydrogen folder. The file must be in .txt format, so you'll need to change its extension. In Finder, rename the file to end with .txt, for example, change "script.rtf" to "script.txt". When a pop-up asks if you want to use .txt, click "Use .txt".

No HTTP server on ports 6969–7069.

A port error usually means you haven't entered a key. To fix this, close both Roblox and Hydrogen. Then, open Roblox. And a pop-up asking for your key should appear before the app opens. Paste your key inside it, and Roblox will open normally. After that, you can then open Hydrogen and execute scripts freely.

How to remove Hydrogen?

To remove Hydrogen from your Mac, you can use the script below.

cd && curl -LJs "https://raw.githubusercontent.com/nullstate9/roblox-stuff/refs/heads/main/executors/hydro-uninstaller.sh" -o "./hydro-uninstaller.sh" && chmod +x hydro-uninstaller.sh && sudo ./hydro-uninstaller.sh

Keep in mind that the script will keep the Hydrogen folder inside your home directory. It reinstalls the normal version of Roblox and removes the Hydrogen app. You can use it to play safely if Hydrogen is detected. If you want a full manual uninstall, delete both Roblox and Hydrogen, along with their folders, and then reinstall Roblox from the website.

Is my Mac supported?

Both Intel and Silicon Macs are supported, as long as they are running macOS 11 or newer (Yes, even Tahoe). You can check your version by typing sw_vers in the Terminal app.

Is Hydrogen Supported on Mobile or Windows devices?

Currently, Hydrogen is supported on macOS only. iOS and Windows versions are being developed. The Android version has been discontinued.

Permission denied (os error)

Check if you are an administrator

  1. Open Settings and go to “Users & Groups”
  2. Check if your account is listed as an Admin

Give Terminal Full Disk Access

  1. Open Settings and go to “Privacy & Security”
  2. Go to “Full Disk Access”
  3. Press the plus icon at the bottom
  4. Add Terminal to the list

If that didn’t fix the error make a ticket.
